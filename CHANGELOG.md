# 更新日志

这个文件记录项目每个正式版本的变化。它面向用户和维护者，而不是只给程序员看。

## v1.3.0 - 2026-06-10

V8：版本发布系统。

- 新增版本中心。
- 新增 `version.json`，让应用能读取最新版本信息。
- 新增检查更新按钮。
- 新增 V8 发布系统学习文档。
- 解释 artifact、release、changelog、版本号和回滚。

## v1.2.0 - 2026-06-10

V7：Supabase 真实云端同步。

- 接入 Supabase 云端账号。
- 新增云端上传和恢复。
- 新增数据库建表 SQL 和 RLS 权限规则。
- 修复云端账号退出后的数据空间切换。

## v1.1.0 - 2026-06-09

V5-V6A：桌面版基础和本地模拟登录。

- 新增 Electron 桌面壳。
- 新增 GitHub Actions 桌面构建。
- 新增本地模拟登录和用户数据隔离。

## v1.0.0 - 2026-06-09

初始可用版本。

- 新增 AI 学习词典页面。
